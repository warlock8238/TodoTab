========================================
           待办提醒助手 Chrome 插件
========================================

📁 项目文件夹：待办提醒助手
📅 创建时间：2024年
🔧 技术栈：Chrome Extension Manifest V3

📋 文件清单：
├── manifest.json          # 插件配置文件
├── newtab.html           # 新标签页HTML
├── newtab.css            # 新标签页样式
├── newtab.js             # 新标签页逻辑
├── popup.html            # 弹出窗口HTML
├── popup.css             # 弹出窗口样式
├── popup.js              # 弹出窗口逻辑
├── background.js         # 后台脚本
├── icons/                # 图标文件夹
│   ├── icon.svg          # SVG图标源文件
│   └── README.md         # 图标创建说明
├── README.md             # 项目说明文档
├── 安装指南.md           # 详细安装指南
├── 快速安装.bat          # 快速安装脚本
└── 项目信息.txt          # 本文件

🚀 快速安装：
1. 双击运行"快速安装.bat"
2. 按照提示完成安装

📖 详细说明：
- 查看 README.md 了解功能特点
- 查看 安装指南.md 了解详细安装步骤
- 查看 icons/README.md 了解图标创建方法

✨ 主要功能：
- 新标签页显示待办事项
- 快速添加和管理待办
- 数据统计和导出
- 系统通知提醒

🎯 使用场景：
- 日常任务管理
- 工作待办提醒
- 学习计划跟踪
- 生活事项记录
